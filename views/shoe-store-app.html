Not Really A Shoe Store

The Not Really A Shoe Store application is a complete full-stack e-commerce web application that has all of the required functionality to be an e-commerce application. It is a full-stack web application built with the NodeJS programming language using express.js, MongoDB for persitant storage, nodemailer for sending conformation email, passport.js for login, authentication, and authorization, and custom styles and features that make this application one of the best and most complex that I have built to-date
